## 当前判断

- StreamingBench real 分数较高，主要低点是 Counting。
- SQA `41.20%` 偏低，之前初步怀疑是历史 QA context / memory 对身份、OCR、局部细节的保存不足，需要继续看 trace。
- Anomaly / Misleading 约 `60-62%`，更依赖视觉细节与短时状态辨别，也需要继续看错误样本。
- Omni 这里只跑了两个视觉任务，没有跑需要音频的完整 omni。
