# 实验跑分

本文件只记录跑分、运行口径和可比性判断。运行命令放在 `08-commands-and-tools.md`，过程性日志放在 `experiment-log.md`。

## 记录规则

- `full` 和 `subset` 不能直接横比。
- `task macro` 是按 task accuracy 平均，`sample weighted` 是按样本数加权。
- `SimpleStream recent4` 表示只看最近 4 帧，不使用 `note / bridge / memory`。
- `plain` 表示直接问模型，不走 StreamWeave 协议。
- 标记为 `旧 adapter` 的 `StreamWeave V3` 结果来自 `2026-04-28` OVO adapter 修复前，只能用于诊断，不作为后续正式结论。
- `smoke / debug` 只说明链路是否跑通，不进入主对比。
